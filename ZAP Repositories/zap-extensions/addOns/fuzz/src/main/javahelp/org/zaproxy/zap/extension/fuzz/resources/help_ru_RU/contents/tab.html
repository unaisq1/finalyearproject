Вкладка «Фаззер»

На вкладке Fuzzer отображаются запросы и ответы, выполняемые при фаззинг a message.
Select a row to see the full requests and responses. You can also search for strings in the fuzz results using the 'Search' tab.

Результаты HTTP-фаззера

Результаты должны быть оценены вручную, чтобы узнать, была ли обнаружена какая-либо уязвимость.

Меню правой кнопки мыши

Щелчок правой кнопкой мыши по строке вызовет меню с теми же параметрами, что и вкладка «История». Плюс предоставление возможности добавлять сообщения из результатов фаззинга на вкладку «Дерево сайтов и история». (сообщения будут помечены FromFuzzer и сначала показать значок фаззера в дереве сайтов).
